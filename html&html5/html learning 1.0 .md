#HTML learning 1.0
1.  标题
`<h1>-<h6>`
2. 水平线
 `<hr></hr>`
3. 注释
` <!---->`
4. 段落
`<p></p>`
5. 折行
`<br></br>`
6. 链接
`<a></a>`
7. 头部
```
<head>头部,可包含 <title>, <style>, <meta>, <link>, <script>, <noscript>, and <base>.
<title>标题
<base>所有链接的默认链接地址
<meta>html文档描述
```
8. 图像
`<img src="" alt=""></img>`
9. 表格
````
<table>
	<thead><tr><th></th></tr></thead>
	<tbody><tr><td></td></tr></tbody>
	<tfoot><tr><td></td></tr></tfoot>
</table>
````
ps: rowspan 和 colspan可以合并表格
10. 表单
```
<ol><li></li><ol>
<ul><li></li></ul>
```
11. 区块
 `<div><span></span></div>`
12. 表单
 ```
 <form>
 <input  />
 <textarea></textarea>
 </form>
```
13. 框架,可以再浏览器窗口显示多个页面
`<iframe></iframe>`
